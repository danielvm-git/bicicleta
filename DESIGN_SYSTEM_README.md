## 📊 Decision Matrix

**Ask yourself:**

1. **What's my brand's personality?**
   - Premium tech? → Sport-Tech
   - Bold statement? → Brutalist
   - Professional? → Editorial
   - Playful? → Retro-Futurism
   - Community? → Organic

2. **Who's my audience?**
   - Tech-savvy? → Sport-Tech
   - Designers? → Brutalist
   - Professionals? → Editorial
   - Gen Z / Gamers? → Retro-Futurism
   - Eco-conscious? → Organic

3. **What will make them return?**
   - Premium experience? → Sport-Tech
   - Bold identity? → Brutalist / Retro-Futurism
   - Trustworthy feel? → Editorial / Organic
